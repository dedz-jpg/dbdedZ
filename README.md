## Projeto — Análise de Dados Imobiliários

Buscando aprimorar meus conhecimentos em SQL e modelagem de dados, iniciei o desenvolvimento de um banco de dados relacional voltado para a análise de vendas imobiliárias. O objetivo do projeto é permitir o acompanhamento de indicadores como VSO (Volume de Vendas sobre Oferta), ticket médio, preço médio por metro quadrado e performance de corretores.

O banco de dados será estruturado de forma a possibilitar a integração com planilhas no Excel e com o Power BI, viabilizando futuras análises de dados, criação de dashboards e extração de insights estratégicos.

As atualizações e evoluções do projeto serão documentadas ao longo do desenvolvimento.

### Tecnologias utilizadas
- MySQL
- SQL
- Excel
- Power BI

### Indicadores analisados
- VSO
- Ticket médio
- Preço médio por m²
- Performance de corretores

### Próximas etapas
- Criação de views analíticas
- Otimização de consultas
- Integração com Power BI
